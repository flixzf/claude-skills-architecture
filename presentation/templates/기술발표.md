# 기술 발표 프레젠테이션 템플릿

이 템플릿은 pptx 스킬을 사용하여 PowerPoint 형식의 기술 발표 자료를 작성하기 위한 참고 자료입니다.

## 슬라이드 구성 (총 15-25장)

### 슬라이드 1: 표지
**레이아웃**: 제목 전용
**내용**:
- 발표 제목 (기술명 또는 주제)
- 발표자명 / 소속
- 발표일
- 컨퍼런스/세미나명 (해당 시)

**디자인**:
- 기술 관련 배경 이미지
- 모던하고 기술적인 느낌
- 제목: 36pt, 굵게

---

### 슬라이드 2: 발표자 소개
**레이아웃**: 제목 + 2단 구성
**내용**:
```
발표자 소개

[프로필 사진]        • 이름: [이름]
                    • 소속: [회사/부서]
                    • 경력: [주요 경력]
                    • 전문 분야: [분야]
                    • 연락처: [이메일]
```

**디자인**:
- 프로필 사진 + 정보
- 간결하고 전문적으로

---

### 슬라이드 3: 목차

**내용**:
```
목차

1. 기술 개요
2. 배경 및 필요성
3. 핵심 기술 설명
4. 기술적 특징
5. 적용 사례
6. 성과 및 효과
7. 향후 계획
```

**디자인**:
- 아이콘 활용
- 기술 관련 색상 (파랑, 청록)

---

**디자인**:

## 스타일 가이드

### 색상
- **주 색상**: 파랑 (#0066CC) - 기술, 신뢰
- **보조 색상**: 청록 (#00A896) - 혁신
- **강조 색상**: 주황 (#FF6600) - 중요 포인트
- **코드 배경**: 진한 회색 (#2D2D2D)

### 폰트
- **제목**: 32pt, 굵게
- **부제**: 24pt, 보통
- **본문**: 20pt, 보통
- **코드**: Consolas 또는 Courier New, 16pt

### 다이어그램
- 아키텍처 다이어그램은 명확하게
- 화살표로 데이터 흐름 표시
- 레이어별 색상 구분
- 범례 포함

### 코드 표시 (필요시)
```python
# 코드 예시
def example_function():
    return "Hello World"
```
- 배경: 진한 회색
- 문법 강조 (Syntax Highlighting)
- 줄 번호 표시
- 간결하게 (10줄 이내)

## 발표 팁

### 기술 발표 특화
- 청중 수준 파악 (전문가 vs 일반)
- 전문 용어 최초 사용시 설명
- 라이브 데모 준비 (가능하면)
- 백업 영상 준비 (데모 실패 대비)
- 성능 지표는 구체적으로

### 시간 배분 (30분 발표 기준)
- 도입 (5분): 배경 및 개요
- 핵심 (15분): 기술 설명 및 특징
- 사례 (5분): 적용 사례
- 결론 (3분): 요약 및 향후 계획
- Q&A (7분): 질의응답
