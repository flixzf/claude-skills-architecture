**레이아웃**: 제목 + 본문
**내용**:
```
BIM (Building Information Modeling)이란?

정의:
건축물의 생애주기 전반에 걸쳐 발생하는 모든 정보를
3차원 모델 기반으로 통합 관리하는 프로세스

BIM ≠ 3D 모델링
BIM = 3D 모델 + 정보 + 프로세스
```

**디자인**:

### 슬라이드 9: BIM의 필요성
**레이아웃**: 제목 + 본문
**내용**:
```
왜 BIM이 필요한가?

기존 방식의 문제점:
✗ 2D 도면의 한계
✗ 정보 분산 및 유실
✗ 의사소통 오류
✗ 설계 변경 시 반영 어려움
✗ 간섭 발견 지연

BIM의 해결책:
✓ 3D 시각화
✓ 통합 정보 관리
✓ 명확한 의사소통
✓ 자동 업데이트
✓ 사전 간섭 체크
```

**디자인**:
- 문제점(빨강) vs 해결책(초록)
- 2단 구성
- 명확한 대비
