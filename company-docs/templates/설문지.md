## 설문 문항 유형

### 1. 객관식 (단일 선택)
- 하나의 답만 선택 가능
- □ 또는 ○ 기호 사용

### 2. 객관식 (복수 선택)
- 여러 답 선택 가능
- □ 기호 사용 + "복수 응답 가능" 명시

### 3. 척도형 (리커트 척도)
- 5점 또는 7점 척도
- 보통, 중립 옵션 포함 여부 결정

### 4. 순위형
- 우선순위 또는 중요도 순서 표시
- ( ) 괄호 안에 순위 기입

### 5. 개방형
- 자유롭게 의견 작성
- 충분한 답변 공간 확보

### 6. 매트릭스 (표 형식)
- 여러 항목을 동일한 척도로 평가
- 표 형식으로 깔끔하게 정리

## Word 문서 작성 시 스타일 적용

- **설문 제목**: 제목 1 스타일, 굵게, 중앙 정렬
- **섹션 제목**: 제목 2 스타일, 굵게
- **질문 번호**: 굵게, 앞에 번호 명시
- **선택지**: 들여쓰기, □ 또는 ○ 기호
- **표**: 표 스타일 적용, 테두리 명확히
- **안내문**: 회색 또는 이탤릭체로 강조
- **여백**: 답변 공간 충분히 확보

## 설문 설계 원칙

1. **명확한 질문**: 이해하기 쉬운 표현 사용
2. **적절한 길이**: 응답 부담을 고려한 문항 수
3. **논리적 순서**: 일반→구체, 쉬움→어려움
4. **중립적 표현**: 유도 질문 지양
5. **일관된 형식**: 비슷한 문항은 같은 형식으로
